This module adds RDF datatype extensions for ARQ constructs, such as queries, expressions and bindings.

```xml
<dependency>
  <groupId>org.aksw.jenax</groupId>
  <artifactId>jenax-arq-plugins-datatype</artifactId>
</dependency>
```
